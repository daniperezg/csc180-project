AI Threat Modeling Agent

Describe your system in natural language. The agent will extract components, data flows, generate a STRIDE-based threat list, and build a draw.io diagram.

Need inspiration? A React frontend calls a Flask API. The API reads and writes from a MongoDB database. There is an admin dashboard for managing users and changing their roles. Authentication is done with JWTs stored in localStorage. The app is deployed on a single VM exposed to the internet over HTTPS.
